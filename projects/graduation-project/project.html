Invizo: Arabic Handwritten Document OCR Solution

Project Overview

Invizo is an open-source Arabic Handwritten OCR Solution designed to perform batch recognition on real-world handwritten Arabic documents. The system operates based on a defined template image of a document type and extracts text fields efficiently. Invizo is powered by a DBNet detection model trained on Arabic Handwritten Line Segmentation. The recognition component is based on a CNN-Transformer model, trained using public handwritten datasets and a custom synthesized dataset to improve recognition accuracy.

Software Versions

1. Enterprise Version

  • Mobile Application – Designed for workers to capture documents.
  • Backend Solution – AI-powered service with storage and interaction capabilities.
  • Desktop Application – For operators to review and refine OCR results.

2. Portable Version

  • Standalone Desktop Application – An all-in-one OCR solution.
