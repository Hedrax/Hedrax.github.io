AI 3D AutoEncoder

Project Details

The Project is about 2D:3D autoencoder to convert 2D images into 3D voxel grid representation. The model consists of encoder, decoder, refiner, and merger for multiple images of one object.
The project follows the PIX2VOX research paper.
The final results are plt files of the best weights model extracted and optimized using PyTorch-Mobile for mobile application local deployment perpose and can be found in the GitHub repo.
